Tabs

The tabs module demonstrate the usage of two components bundled into a single module. The tabs itself use the <angular-tabs> selector. The structure is like this:

<angular-tabs>
<angular-tab title="Title of the tab">
This ist the content of the tab!
</angular-tab>
</angular-tabs>

Calculator

The calculator component shows an input field with an attached calculator. This looks like this:

<angular-calculator></angular-calculator>

You can also use your num pad to enter numbers.

Tasklist Store

The Tasklist Store implements the Datastore Pattern by separating backend logic and view state. The component binds to a service variable and display the current state of that variable. That service variable is managed by a datastore and never changed directly. The datastore accepts several actions (like 'LOAD' or 'DELETE') and applies those actions to the view state. A in memory mock implementation is used here to mimic a rest backend. Currently only the SELECT and DELETE actions are implemented in the view.

A generic store version is provided, too.
